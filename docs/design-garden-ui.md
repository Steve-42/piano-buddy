# 花园 UI 设计（MVP）

## 目标
将练习页面从工具感的计时界面改为温暖的花园体验，通过植物成长可视化练琴积累。

## 成长阶段

| 累计弹奏 | 阶段 | 视觉 |
|----------|------|------|
| 0 分钟 | 种子 | 🌰（小） |
| 5 分钟 | 发芽 | 🌱（中） |
| 15 分钟 | 成长 | 🌿（大） |
| 30 分钟 | 开花 | 🌸（最大） |

植物用 emoji 表示，通过 CSS 缩放体现成长（如 1x → 1.5x → 2x → 3x）。
阶段之间平滑过渡动画。

## 页面改动

### 练习页面（PracticeView）— 主要改动
- **背景**：温暖色调的花园图片 + 半透明蒙版
- **中央**：植物 emoji，随累计时间成长
- **弹奏时**：音符 emoji（🎵）从上方缓慢飘落，像雨露浇灌
- **计时器**：保留但弱化，移到底部或顶部小字显示
- **停止按钮**：保持可见但不抢视觉焦点

### 首页（HomePage）— 轻微调整
- 色调从深色冷调调整为温暖色调
- 显示当前植物状态（今日已练到什么阶段）

### 其他页面
- 暂不改动，保持一致的暖色调即可

## 色调方向
- 当前：深蓝灰（slate-800/900）
- 目标：温暖明亮（amber/emerald/warm tones），联想到阳光下的花园
- 背景：浅绿或暖黄渐变，或一张柔和的花园图片

## 音符飘落动画
- 检测到琴声时，随机位置生成 🎵 emoji
- 缓慢下落（CSS animation，约 3-4 秒落到植物位置）
- 同时存在 3-5 个音符，营造"浇灌"的感觉
- 无琴声时音符停止生成，已有的自然落完

## 自定义背景
- 设置页新增"背景图片"选项，用户可从本地选择图片
- 图片转为 data URL 存储在 IndexedDB（UserSettings 新增字段）
- 未设置时使用默认的渐变背景

## 不做（MVP 范围外）
- 多种植物可选
- 花园累计展示
- 雨露/阳光切换效果
- 练习结束的成长回放动画
- 改名
